## Quick Start

1. Clone the repository:

```sh
git clone https://github.com/dhunanyan/aristobyte.git
cd aristobyte
```

2. Use the Node version from `.nvmrc`:

```sh
nvm use
```

3. Install dependencies:

```sh
yarn install
```

4. Start the dev server:

```sh
yarn dev
```

Open `http://localhost:3000` in your browser.

## Scripts

- `yarn dev` — Start the development server
- `yarn build` — Create a production build
- `yarn start` — Run the production server
- `yarn lint` — Run ESLint

## Project Structure

- `app/` — Routes, layouts, and global styles
- `components/` — Reusable UI components
- `config/` — Navigation and page configuration
- `context/` — Providers and hooks
- `data/` — Translations and metadata
- `public/` — Static assets
- `styles/` — Global SCSS settings and mixins
- `utils/` — Utilities and shared constants
